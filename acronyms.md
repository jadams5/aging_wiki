# Acronyms

Reference glossary for acronyms and abbreviations used across the wiki. Curated — common-knowledge terms (DNA, RNA, ATP, PDF) included only when colliding with a wiki-specific use; obvious one-off mentions omitted.

When in doubt about a term, search this page first, then the relevant atomic page (e.g., `[[mtor]]` for mTOR-pathway detail).
