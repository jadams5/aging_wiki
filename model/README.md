# model/ — canonical v0.3 aging-simulator engine

`engine.mjs` is a dependency-free, pure-ESM reimplementation of the verified **v0.3** aging-simulator model: per-node intrinsic burden curves, bounded fixed-point coupling across the 18-node / 38-edge hallmark causal graph, per-sex CDC-WONDER-2022 competing-hazards mortality (no sex scalar), survival / life-expectancy integration, the six-way per-age hazard decomposition, and node-freeze / lifestyle interventions. The parameter source of truth is [`frameworks/causal-graph-parameters.md`](../frameworks/causal-graph-parameters.md) (its canonical `json` block); `build-params.mjs` extracts that block to `params.json`, which the engine, tests, and CLI consume. Run the build with `node model/build-params.mjs`, the regression suite with `node model/test.mjs` (or `node model/cli.mjs validate`), and the CLI with e.g. `node model/cli.mjs le --sex female` / `node model/cli.mjs sweep --node atherosclerosis`. The interactive app `viz/aging-simulator.html` currently still embeds its own copy of this math; refactoring it to import this engine is a separate follow-up — until then `test.mjs` pins the exact v0.3 targets so the two implementations cannot silently diverge.
